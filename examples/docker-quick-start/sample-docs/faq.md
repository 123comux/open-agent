# Acme Corp FAQ

## General

### Q: What is Open Agent?
A: Open Agent is an open-source Agentic RAG autonomous work assistant that
combines retrieval-augmented generation with tool-using agents to help you
chat with your own knowledge base, automate tasks, and ship LLM workflows.

### Q: Which models are supported?
A: OpenAI, Anthropic Claude, Ollama, and Zhipu AI. Any OpenAI-compatible
endpoint can be configured via `OPEN_AGENT_BASE_URL`.

### Q: What's the license?
A: MIT License.

## Pricing

### Q: Is Open Agent free?
A: Yes, open-source under MIT. You pay only for your LLM API usage.

### Q: Do you offer cloud hosting?
A: Not yet. Self-host via Docker or pip.

## Support

### Q: How do I report bugs?
A: Open an issue on GitHub: https://github.com/123comux/open-agent/issues

### Q: Is there a Slack/Discord?
A: Not yet, use GitHub Discussions.
